# Task02

Простое веб-приложение на PHP. Игроку показываются два случайных числа, он вводит своё имя и наибольший общий делитель. Программа проверяет ответ и сохраняет результат в базу данных SQLite.

## Запуск

Находясь в папке Task02, выполнить:

    php -S localhost:3000 -t public

Открыть в браузере http://localhost:3000

## Файлы

public/index.php — главная страница с игрой  
public/log.php — история всех игр  
db/ — сюда создаётся файл базы данных (data.db появится сам при первом запуске)
